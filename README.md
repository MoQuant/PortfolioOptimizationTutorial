# PortfolioOptimizationTutorial

In order for the stock data function to work properly, you must make an account with FinancialModelingPrepAPI in order to get an api key. Other than that the other API should work well. For this tutorial you will need to have numpy, pandas, and matplotlib installed, along with requests.

## Stock API Key Required
In order for this program to work, you must create a free account on Financial Modeling Prep and get an api key to paste in the Stocks.py file.<br/>
